Instrucciones de implementación

1 Añade el script universal de MailerLite

Pega este código justo antes de la etiqueta </head> de tu web:

<!-- MailerLite Universal -->
<script>
    (function(w,d,e,u,f,l,n){
      w[f]=w[f]||function(){(w[f].q=w[f].q||[]).push(arguments);},
      l=d.createElement(e),l.async=1,l.src=u,
      n=d.getElementsByTagName(e)[0],
      n.parentNode.insertBefore(l,n);
    })(window,document,'script',
      'https://assets.mailerlite.com/js/universal.js','ml');
    ml('account', '2170745');
</script>
<!-- End MailerLite Universal -->

2 Añade el script de integración

Pega esto justo antes de </body>:

<script src="mailerlite-integration.js"></script>

O si prefieres no usar un archivo externo, copia todo el contenido del archivo mailerlite-integration.js dentro de una etiqueta <script>.

3 Verifica que tu formulario tenga las clases correctas

El script busca estos elementos en tu HTML:

<div class="nl-form">
  <input class="email-input" type="email" placeholder="...">
  <button class="subscribe-btn">Suscribirme ahora</button>
</div>

Si tus clases CSS son diferentes, actualiza los selectores al principio del archivo JS.

4 Activa el formulario en MailerLite

Ve al dashboard de MailerLite y asegúrate de que el formulario "Newsletter — Sara Vrai" esté activo.

Qué pasa cuando alguien se suscribe

1. El visitante escribe su email y pulsa "Suscribirme ahora"

2. El script envía el email a MailerLite

3. Aparece un mensaje de confirmación en la web

4. El suscriptor se añade automáticamente al grupo "Lista de Espera — Sara Vrai"

5. Se dispara el correo de bienvenida (si está configurado en Automations)

Nota: El formulario tiene el double opt-in desactivado, por lo que los usuarios entran directamente a la lista sin tener que hacer click en un correo de confirmación previo. Asegúrate de tener una automatización (workflow) de bienvenida configurada para recibirlos.
